## 1. ⚠️ 头号约束：全吸收深度标准（HIGHEST PRIORITY）

### 1.1 问题（作者 2026-06-16 反馈，已核实）
作者指出"每一章内容有点少，只是凝练，没把三书所有内容融入；原始章节一章可三五十页"。**实测确认属实**：

| 成稿章（`.tex` 行） | 吸收的源章（MD 行） | 吸收比 |
|---|---|---|
| 非线性优化 554 | 十四讲06(1027)+Barfoot03(2072)+Barfoot04(824)+Handbook01(632) | **~1/6** |
| 状态估计 667 | Barfoot01(1227)+Barfoot02(1983)+Handbook01(632) | **~1/6** |
| 卡尔曼/ESKF 579 | Barfoot03(2072)+Barfoot04(824)+十四讲09(1098) | **~1/7** |
| 相机模型 453 | 十四讲05(732)+Handbook07(460)+Barfoot08 部分 | **~1/3** |

单是十四讲 ch6（1027 行）一本就近乎整章 2 倍，而该章号称吸收四个源章。**确属凝练 2–6 倍。**

### 1.2 病根
综合成章时优先了 v5.0 教学模板叙事，把推导/例题/表**摘要**而非**全录**；教学脚手架（自测/导航/小结）占了页面，技术质量被压缩。

### 1.3 铁律（今后强制）
1. **全量独立记录**：每章须把所覆盖各源章的 **每一步推导（中间代数不跳）、每一道例题与数值例、每一条定理/引理/命题 + 完整证明、每一张表/分类/算法伪码** 完整写入。读者**不必再翻原书**。
2. **跨书同题两书视角都记**：并列复述 + 对比（标注谁更严谨/谁更直觉），**绝不择一带过**。
3. **零外部 punt**：`\cite` 仅标出处；**严禁**"详见 Barfoot §8""见原书""闭式见十四讲§4.5"之类把内容推给原书的写法（具体待清理项见 §10.3）。
4. **教学脚手架是增量**：v5.0 模板（§6.2）的自测/导航/陷阱/小结**叠加在**完整技术内容之上，不替代它。
5. **目标页数 ≈ 所替代源章总量**：作者认可一章 **30–50 页**。不要为"显得精炼"而压缩。

### 1.4 如何自检（执行者必跑）
```bash
# (a) 外部 punt 排查——应为空
grep -rnE '详见|参见|见十四讲|见原书|见 ?Barfoot|见 ?Handbook|闭式.{0,8}见|此处略|从略|不再赘述|略去' parts/

# (b) 吸收比粗估——我方章 .tex 行数 应与源 MD 行数同量级（非 1/5）
wc -l parts/<部>/<章>.tex
wc -l ../Barfoot_SER2_md/<章>/<章>.md ../视觉SLAM十四讲_md/<章>.md
```
逐节核对：拿源章目录，逐小节确认"每条定理/推导/例/表"在我方章是否都有对应、完整的独立记录。缺则补。


**架构铁律**：
- **唯一全局定义处 = `common/preamble.tex`**（含 `\documentclass[lang=cn, 10pt, device=normal]{elegantbook}` + 全部宏包/宏/cleveref/`\input{styles}`）。**改字体/全局选项只动这一处**；各驱动（book/vol_*）首行 `\input{common/preamble}`。
- **章序不硬编码**：顺序 = `part.tex` 里 `\input` 的先后；章号由 `\chapter` 自动生成。增删章 = 增删 `\input` 行。
- **章标签 = 既有短名**（见各 `.tex` 第 2 行 `\label{ch:xxx}`，**非文件名**）。跨章一律 `\cref{ch:xxx}`，**禁手写章号/式号**。
- **分层**：`parts/Pk_xxx/part.tex` 含 `\part{}` + 顺序 `\input{parts/Pk/章}`（根相对路径）。
- **分册双构建**：经 `xr-hyper` + `\externaldocument{build/book}` 跨卷 `\cref` 落到 book；`compile.sh` 编译分册时先建 book 出 `book.aux`。
- **工程不单列部**：按章并入各章末「实践」节（仿十四讲，如刚体章 §Eigen、李群章 §Sophus）。

## 6. 写作规范（摘要；全文见 `docs/编写规范.md`）

### 6.1 v1.0 四锁决策（冻结，改走版本号）
| 决策 | 选定 |
|---|---|
| 推导位置 | **证明优先内联正文**（主推导一律正文）；**仅极冗长/纯机械补充**（如 SE(3) 的 Q 块闭式）入章末 `derivation` 附录或脚注 |
| 引用粒度 | **逐公式**：每条非平凡公式/结论在出现处 `\cite` 源 |
| 扰动方向 | **右扰动为主** $\mathbf R\,\mathrm{Exp}(\delta\boldsymbol\phi)$（局部坐标，对接 GTSAM/Sophus/因子图/自动微分）；引用 Barfoot 左扰动处并列给出转换 |
| 深度基线 | **复现级 + 不证已假定背景**：每步可独立复现；线代/微积分/概率不重证；超纲严格性入脚注/附录 |

### 6.2 v5.0 教学模板（上位标准，循序渐进·展开思路）
理论教学章结构：**前置自测 → 本章目标 → 知识导航(tikz) → 前置桥接 → 如果跳过会怎样 →〔各节：动机 → 反面（不这样会怎样）→ 历史 → 理论 → 陷阱(pitfall) → 练习(practice)〕→ 常见误解汇总 → 小结（符号表 + 定理速查表 + 知识点总表）→ 延伸阅读 → 后续关系 → 故障排查 → 推导附录(derivation, 精简)**。
- 十五原则 R1–R15（深度优先/循序渐进/图表结合/实践/先动机后理论/多视角认知工具/语言精确标边界/文字为主代码为辅/陷阱专栏/难度⭐/累积项目/前置自测/跨章桥接/故障诊断）。
- 五质量闸 G1 规范→G2 事实→G3 一致→G4 完整→G5 润色。**禁元叙事词**：worker/审稿/复检/TODO/待补充/占位 等不得出现在正文。
- 理论章 text:code ≥ 85:15。**不按行数判定**（LaTeX 更密），以"模板各节齐全 + 展开深度 + §1 全吸收"判定。
- **注意**：v5.0 教学层与 §1 全吸收标准**并行满足**——既要展开思路，又要全量记录技术内容。二者不是二选一。

### 6.3 学术化/体系化要点
- 概念→`definition`；论断→`theorem/lemma/proposition`，条件结论分明；no orphan formula。
- 每符号首次出现即定义；全书统一记号表（§7）。
- 假设显式（线性化有效域、可观性、噪声假设、收敛条件）。
- 全程严格携带协方差/信息矩阵（吸收 Barfoot 严谨，补到十四讲式叙述）。
- 图一律 **tikz 重画**（版权合规 + 风格统一），不搬原图。
- 严格依赖排序：目录即知识依赖图，后用概念不先于定义出现。

### 6.4 论文集成三模式（按优先）
1. **正文成节（主）**：论文核心写成正文小节，按主题组织，`\cite` 标源。
2. **抽取片段（inline）**：结果嵌入已有小节。
3. **精读盒（按需）**：`paper` 盒，固定结构 问题→核心思想→关键结果→与本书关系→局限。
- 文献管理：统一 `refs.bib`；键名 `作者年份关键词`（如 `qin2018vins`）；经 Zotero(MCP)/`/academic` 技能 形成 论文→Zotero→精读→集成 流水线。

## 7. 锁定记号与约定（全书强制）

- **旋转** $\mathbf R\in\mathrm{SO}(3)$（Barfoot 用 $\mathbf C$，引用处转换）；**位姿** $\mathbf T\in\mathrm{SE}(3)$。
- **四元数 Hamilton**（与 Eigen/十四讲一致），引用 JPL 处显式转换。
- **扰动右扰动为主** $\mathbf R\,\mathrm{Exp}(\delta\boldsymbol\phi)$，右雅可比 $\mathbf J_r$ 为主；Barfoot 左扰动处并列转换。
- **$\mathfrak{se}(3)$ 排序** $\boldsymbol\xi=[\boldsymbol\rho;\boldsymbol\phi]$（平移在前），且 $\boldsymbol\rho\neq\mathbf t$（相差左雅可比 $\mathbf J$）。
- **坐标系**下标语义统一（如 $\mathbf T_{wc}$ = 相机到世界），主动/被动声明。
- **噪声协方差** $\boldsymbol\Sigma_{\mathbf w}$（过程）/ $\boldsymbol\Sigma_{\mathbf v}$（观测）——避开 Barfoot(Q/R) 与十四讲(R/Q 反向) 的字母冲突，且把 $\mathbf R$ 留给旋转。**信息矩阵** $\boldsymbol\Lambda=\boldsymbol\Sigma^{-1}$、**信息向量** $\boldsymbol\eta$；**先验** $\check{\mathbf x}$ / **后验** $\hat{\mathbf x}$（Barfoot check/hat）。
- **残差 Jacobian** $\mathbf J=\partial\mathbf e/\partial\mathbf x$；目标函数梯度 $\nabla J$、Hessian $\mathbf H_J$；GN 统一 $\mathbf H=\mathbf J^\top\mathbf W^{-1}\mathbf J$（标注十四讲 $\mathbf J\mathbf J^\top$ 标量约定）。
- **公式编号** `\numberwithin{equation}{chapter}` → 式 章.序；`secnumdepth=2`+`tocdepth=1`。
- **代码框** `\begin{codebox}[语言]{标题}`：可选 #1 = listings 语言名（C++/Python/bash…），**仅驱动语法高亮**（右上角语言角标已移除——非官方）；留空则无高亮。
- **代码注释禁 Unicode 数学符号**（等宽字体 Latin Modern Mono 缺 ᵀ⁻¹ηΛΣδξ← 等字）→ 一律 ASCII（`^T`/`^-1`/`Lambda`/`delta_xi`…）；**中文注释 OK**。
- 批注宏：`\pz`（黄）/ `\pzr`（红）/ `\rebuilt`（橙，重建待核对）。
