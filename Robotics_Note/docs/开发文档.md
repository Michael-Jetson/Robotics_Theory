# 《机器人学笔记》开发文档

> ElegantBook v4.7 官方 API（实测自 cls 源码）+ 本项目排版约定 + 版式基线原则。
> 基于 TeX Live 2026 内 `elegantbook 2026/05/01 v4.7`（`/usr/local/texlive/2026/texmf-dist/tex/latex/elegantbook/elegantbook.cls`）。编译链 XeLaTeX + biber，经 `compile.sh`。
> 改版式前先读 §1（基线原则）。末次更新 2026-06-16。

---

## 1. ★ 版式基线原则（最重要）

**先全部回归官方 ElegantBook，再基于官方小步微调；勿大改、勿自造 hack。**（作者定）

- **页眉 / 页边 / 字体 / 主题色 / 代码默认风格** —— 全部**沿用官方**，不在 `preamble.tex` 用 `fancyhdr` / `geometry` 覆盖。
  - 教训：曾用 `\geometry{inner=2.2cm, outer=4.0cm}` 覆盖（只设内外边距、**丢了上边距/headsep**）→ **页眉上飘顶到页顶**；曾用 fancyhdr 把页眉改「居中 + 全宽黑线」→ 朴素丢失官方设计感。**均已撤销回官方。**
- **扩展只走官方机制**：新定理类盒用 `\elegantnewtheorem`（§6）；颜色只用主题语义名 `main/second/third/structurecolor`（§4）。
- **禁自造 hack**：`sharp corners`、`overlay` 角标、不对称 `leftrule`/`borderline west`、自造 RGB 色——一律不用。内容盒（§7）也按 EB 圆角细框风格。
- **必要微调**（已在官方上叠加，见各节）：按章公式号、`secnumdepth2/tocdepth1`、cleveref 中文名、自定义内容盒、todonotes 批注、`10pt`。

> 本项目当前 documentclass：`\documentclass[lang=cn, 10pt, device=normal]{elegantbook}`（**color 用默认 = blue 主题**）。全局定义唯一处 = `common/preamble.tex`。

## 2. 编译

```bash
./compile.sh            # 默认 book；分册 ./compile.sh vol_slam|vol_planning|vol_control
```
- 引擎**必须 XeLaTeX**（中文）；`latexmk -xelatex` + biber。
- 源**只读**挂载到容器，中间文件留容器，**仅 `build/<target>.pdf` 回本地**。须**零错、零未定义引用**。
- **biblatex 由类自动加载**（`bibend=biber` 默认）→ **勿再** `\usepackage{biblatex}`；直接 `\addbibresource{}` + `\printbibliography`。
- Bash 调用本仓库 docker 命令需 `dangerouslyDisableSandbox: true`。

| `bibend` | `biber`(默认) / `bibtex` | 文献后端 |

## 4. 颜色系统（官方语义名，随主题变）

四个语义名：`structurecolor`（结构：页眉/章节标题/链接）、`main`、`second`、`third`。各主题 RGB（cls 实测）：

| 主题 | structurecolor | main | second | third |
|---|---|---|---|---|
| **blue（默认，本项目）** | **蓝 (60,113,183)** | **绿 (0,166,82)** | **橙 (255,134,24)** | **青 (0,174,247)** |
| green | 绿 (0,120,2) | 绿 (0,120,2) | 橙 (230,90,7) | 青 (0,160,152) |
| cyan | 青 (31,186,190) | 绿 (59,180,5) | 黄 (175,153,8) | 红 (244,105,102) |
| gray / black | 灰/黑（四色同值，去色）| — | — | — |

- 本项目（blue 主题）盒色映射：**页眉/章节标题 = structurecolor 蓝**；`definition`/`codebox`/`insight` = main 绿；`theorem`/`practice`/`paper` = second 橙；`proposition`/`derivation`/`algo` = third 青；`pitfall` = 红（warning，EB 无红主题色，唯一例外）。
- 固定调色板：`main1–5` `second1–5` `third1–5` `structure1–5`、`winered`（链接）。
- 自定义盒**只用语义名**（如 `colframe=third`、`colback=main!75!black`）→ 自动跟随主题。

## 5. 官方页眉 / 页边 / 字体（**勿覆盖**，见 §1）

**页眉（fancyhdr，cls 1258-1276）**——官方实现，本项目沿用：
```latex
\fancyhead[EL]{\color{structurecolor}\cnormal\leftmark}   % 偶页左：章名
\fancyhead[OR]{\color{structurecolor}\cnormal\rightmark}   % 奇页右：节名
\fancyhead[R]{\color{structurecolor}\cnormal\rightmark}    % oneside：右=节名
\renewcommand{\headrule}{\color{structurecolor}\hrule width\textwidth}  % structurecolor 规则线
\renewcommand{\headrulewidth}{1pt}
\chaptermark→\markboth{章名}{} ; \sectionmark→\markright{\thesection\, 节名}
% 章首页 plain：无页眉、无线。
```
→ 效果 = 双侧章/节跑马灯（structurecolor）+ 1pt 主题色线，版心留白正常。**若想居中**：在 preamble 重覆盖 fancyhdr（注释块留在 preamble），但属偏离官方，慎用。

**页边（geometry，cls 130-147）**：EB 自设含**上边距/headsep**。封面用 `\newgeometry{margin=0in}` 后 `\restoregeometry`。**勿在 preamble 用 `\geometry{}` 覆盖**（漏设上边距→页眉上飘）。

**字体**：`\setmainfont{TeXGyreTermesX}` + `anyfontsize`；CJK 由 EB 按 `chinesefont` 设（默认 fandol）。

## 6. 内置环境（`mode=fancy`）

**彩色定理类（自动「章.序」编号，tcolorbox 彩盒）**

| 环境 | 颜色 | | 环境 | 颜色 |
|---|---|---|---|---|
| `theorem` | 橙 second | | `definition` | 绿 main |
| `proposition` | 青 third | | `lemma`/`corollary`/`axiom`/`postulate` | 橙 second |

**练习类（独立计数）**：`example` `exercise` `problem`
**不编号说明类**（官方实测，非盒，行内加粗彩标题）：`proof`(橙) `solution`(绿) `note`(红⚠+橙名) `remark`(橙) `assumption`(青) `conclusion`(青) `property`(青) `custom{标题}`(青)
**版块**：`introduction[名]` `problemset[名]`（structurecolor 彩盒）
**星号版**（`theorem*` 等）：不编号。

```latex
\begin{definition}[特殊正交群]\label{def:so3} ... \end{definition}   % [..] = 可选注
\begin{theorem}[指数映射]\label{thm:exp} ... \end{theorem}
```
> `note`/`proof` 等是**行内环境**（非彩盒）：`\begin{note} ... \end{note}` 渲染为红⚠ + 橙「笔记」+ 斜体正文。

## 7. 自定义新单元 `\elegantnewtheorem`（官方扩展机制）

```latex
\NewDocumentCommand\elegantnewtheorem{ m m m g O{} }
% {环境名}{显示名}{风格}{引用前缀(可选花括号)}[键值(可选)]
```
风格（第 3 参，对应官方三色）：`defstyle`(绿 main) / `thmstyle`(橙 second) / `prostyle`(青 third)。

本项目（`styles.tex`）：
```latex
\elegantnewtheorem{algo}{算法}{prostyle}{alg}      % 青「算法」盒
\elegantnewtheorem{insight}{洞见}{defstyle}{ins}   % 绿「洞见」盒
\elegantnewtheorem{paper}{论文}{thmstyle}{paper}   % 橙「论文」盒
```
- 生成 `algo` + `algo*`，编号「章.序」，引用 `\cref{alg:标签}`。
- 用法同内置：`\begin{algo}[标题]\label{alg:x} ... \end{algo}`（`[..]` 为可选注，amsthm 风）。
- **底层 amsthm**（`\newtheorem`）→ 计数器名 `tcb@cnt@<env>`，cleveref 须绑定（§10）。

## 8. 本项目自定义内容盒（`styles.tex`，EB 风·无 hack）

EB 无「代码/推导/陷阱/练习」盒，故自造；但**按 §1 用 EB 主题色 + 圆角 + 细框**，零 hack。均走 `[可选标题]` 接口（不破内容）。

| 环境 | 用途 | 颜色 | 用法 |
|---|---|---|---|
| `codebox` | 代码框（标题 + 语法高亮 + 行号）| main 绿 | `\begin{codebox}[C++]{标题} ... \end{codebox}` |
| `derivation` | 推导框（可跨页）| third 青 | `\begin{derivation}[标题] ... \end{derivation}` |
| `pitfall` | 陷阱/思维误区 | 红 warning | `\begin{pitfall}[思维陷阱：…] ... \end{pitfall}` |
| `practice` | 练习/前置自测 | second 橙 | `\begin{practice}[练习] ... \end{practice}` |

- **codebox 的 `[语言]`**（C++/Python/bash…）**仅驱动语法高亮**（`listings` 库 + `notecode` 风格）；**右上角语言角标已移除**（非官方、撞色）。留空则无高亮。
- 依赖 `\tcbuselibrary{listings}`（类已载 tcolorbox+listings 宏包，但 listings 库需手动开）。
- `pitfall`/`practice`/`derivation` 当前**无计数器**（不参与 `\cref`）；若要编号可改走 §7 `\elegantnewtheorem`，但会改变 ~70 处调用语义，属大改，慎。

## 9. 公式 / 章节编号（微调）

```latex
\numberwithin{equation}{chapter}   % EB 默认全书连续 → 改「章.序」式 (3.1)
\setcounter{secnumdepth}{2}        % 只编到 section（避免 \paragraph 出现 2.2.0.0.1）
\setcounter{tocdepth}{1}
```
- 引用：`\label{eq:x}` + `\cref{eq:x}` → 「式 (3.1)」。不编号用 `equation*` / `\notag`。
- 图/表本就按章（book 类默认）。

## 10. 交叉引用（cleveref，不硬编码）

```latex
\crefformat{chapter}{第#2#1#3章}  \crefformat{section}{第#2#1#3节}
\crefformat{equation}{式~#2(#1)#3} \crefname{definition}{定义}{定义} ...
% 定理盒计数器名 tcb@cnt@<env>，须绑定否则报 undefined：
\makeatletter \crefname{tcb@cnt@theorem}{定理}{定理} ... \makeatother
```
- 章序 = `part.tex` 中 `\input` 先后；章号 `\chapter` 自动。**禁手写「第3章」「式(3.1)」**，一律 `\cref{ch:标签}`/`\cref{eq:标签}`。
- 章标签 = 既有短名（`ch:lie`/`ch:nlopt`…，见各 `.tex` 第 2 行），非文件名。

## 11. 脚注 / 旁注 / 代码注释

- **脚注** `\footnote{}`（类载 footmisc）。
- **旁注**（本项目 todonotes）：`\pz{}` 黄批注 / `\pzr{}` 红重点 / `\rebuilt{}` 橙「重建·待核对」/ `\marginnote{}` 纯文字侧注。
- **代码注释禁 Unicode 数学符号**（等宽字体缺 ᵀ⁻¹ηΛΣδξ← 等字）→ 一律 ASCII（`^T`/`^-1`/`Lambda`…）；中文注释 OK。

## 12. 排版坑 / 注意

1. **公式默认不按章编号** → 必须 `\numberwithin{equation}{chapter}`。
2. **biblatex 已被类加载** → 勿重复 `\usepackage{biblatex}`；后端默认 biber。
3. **cleveref** 日志 `First Aid for cleveref.sty applied!` 属正常兼容补丁，非错误。
4. **代码框** 需 `\tcbuselibrary{listings}`（类默认只开 skins/breakable/theorems/fitting/raster）。
5. **cleveref 引定理盒** 须绑 `tcb@cnt@<环境>`（§10）。
6. **勿覆盖 geometry/fancyhdr**（§1/§5）——否则页眉上飘 / 丢官方设计感。
7. 中文必须 XeLaTeX/LuaLaTeX，勿 pdfLaTeX。
8. **唯一全局定义处 = `common/preamble.tex`**（含 documentclass）；各驱动首行 `\input{common/preamble}`。`main.tex` 已删。
