### 1.1 `quad_kin` 单腿逆运动学 θ₁ 数值例符号错（写作 agent 转写漏负号）
- **错误**：FK/IK 数值例代入 $l_2=l_3=+0.27$，算得足端 $(0,0.088,+0.382)$；逆解第一步硬说 $\theta_1=\operatorname{atan2}(0^+,-)\approx0$。实算 $\operatorname{atan2}(0.0672,-0.138)\approx154^\circ\ne0$——例**不自洽**，回代并未真正闭环。
- **根因**：宇树式 5.9–5.10 明确 $l_2=-l_{hip}$、$l_3=-l_{knee}$（**取负**，足端 $z$ 轴朝上、足在下为负）。写作 agent 用了 $+0.27$，使足端 $z$ 反号。
- **订正**（核宇树式 5.9–5.11）：例改用 $l_2=l_3=-0.27$ → 足端 $(0,0.088,\mathbf{-0.382})$（与"腿根正下方"物理吻合）；逆解 $\theta_1=\operatorname{atan2}(0,\,0.1536)=0$ **洁净复原**（$z<0$ 使分子两项 $Ly+l_1z$ 恰好抵消）。FK 例 + IK 例 + 文字说明均已改。

### 1.2 `quad_gait` 堵转保护（anti-windup）系数 ϖ 方向反
- **公式忠实**：凸组合式 $({}_d^{\mathcal O}p^1=\varpi\,{}_d^{\mathcal O}p^0+(1-\varpi)\,{}^{\mathcal O}p^0+v\Delta T)$ 忠实于宪元式 2.62（ϖ 乘**期望/纯积分**位置、$1-\varpi$ 乘**实际**位置）。
- **错误**：`insight` 盒与「故障排查」均把方向说反——称"$\varpi$ 越大→期望越跟随实际、抗饱和越强、对策增大 $\varpi$ 如 0.99"。按公式，$\varpi\to1$ 是**纯积分**（期望脱离实际、windup），恰恰相反。"0.99"源中**未给定**（agent 臆造值）。
- **订正**：补稳态推导——撞墙（实际卡住）时 $\,{}_d^{\mathcal O}p$ 有界停在 ${}^{\mathcal O}p^0+\frac{1}{1-\varpi}v\Delta T$，误差界 $\frac{\|v\|\Delta T}{1-\varpi}$。故 **$\varpi$ 越小→界越紧→抗饱和越强；$\varpi\to1$→界→∞→保护消失**。于宪元原话"$\varpi$ 越大、抵抗外力能力越强"指**自由空间位置刚度**（另一端）。故障排查对策改为「**减小** $\varpi$」。"0.99"标注为示意（源未给值）。

### 1.4 `quad_kin` 欧拉角逆解写作单参数 atan（源真错，已订正）
- **源印**：于宪元式 2.21、2.22 三个角分量逆解用单参数 $\operatorname{atan}$。
- **错因（源真错）**：单参数反正切值域仅 $(-\pi/2,\pi/2)$，无法区分象限，丢失姿态符号信息 → 逆解错。宇树单腿逆解（式 5.18、5.37）即用 atan2，可佐证。
- **正确**：三式一律用双参数 $\operatorname{atan2}(y,x)$（值域 $(-\pi,\pi]$、依分子分母符号定象限），即 `eq:qk-R-to-euler`。正文已为正确版；本轮把原 `[OCR 订正]` 元注释改写为客观约定说明（不提 OCR/源）。
