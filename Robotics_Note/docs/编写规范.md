## 一、学术化（rigor）

1. **形式化**：概念一律 `definition`；论断一律 `theorem/lemma/proposition`，**条件与结论分明**。
2. **完整推导，不跳步（证明优先内联）**：每个非平凡结论都有**完整可查**的推导路径，**主要证明放正文**（教材兼笔记=体系化知识，如旋转矩阵引出 $\so(3)$、Rodrigues 公式证明、雅可比/扰动求导，就近用 `derivation` 盒或 `proof`）；**仅极冗长或纯机械的补充细节**才入章末附录/脚注。**no orphan formula**（不许凭空出现结论）。
3. **精确记号**：全书统一记号表；每个符号**首次出现即定义**。
4. **出处可追溯（逐公式）**：每条结果在其出现处 `\cite` 到源——对标 Barfoot：一节数条公式、每式背后一篇论文，但叙事综合成连贯的“面”（topic-first），而非论文罗列。
5. **假设显式**：每个模型/定理写清**假设与适用范围**（线性化有效域、可观性、噪声假设、收敛条件）。
6. **不确定度一以贯之**：状态估计全程严格携带协方差 / 信息矩阵（吸收 Barfoot 的严谨，补到十四讲式叙述）。
7. **深度基线**：复现级——每步可独立复现；线代/微积分/概率等**假定背景不再重证**；超纲严格性入脚注/附录。
8. **自包含 · 独立记录（铁律，不以引用代替内容）**：吸收三书须把**全部有价值内容完整写入书中**——定义、**每一步推导**、结论、代码、直觉，读者**无需翻原著**即可读懂。`\cite` 仅标**出处/致谢**，**绝不**用「详见 Barfoot §8」「见原书」替代正文内容。三书对同一点的不同视角 → 在书中**各自独立复述并合并讲解**，而非指过去。
9. **网络增补（教学导向，允许且鼓励）**：可结合互联网（WebSearch/WebFetch）吸取**有利于教学**的内容——更好的直觉/历史脉络/反例/可视化思路/最新进展/更清晰表述——用于**补充或优化**讲解，而非堆砌。网络来源同样 `\cite`/标注；重述或存疑打 `\rebuilt`/`\pz` 待核。

## 二、体系化（systematic）

1. **严格依赖排序**：后用的概念不先于其定义出现；**目录即知识依赖图**。
2. **统一章节模板**（见四）。
3. **全交叉引用**：定义/定理/公式/图全部 `\label` + `\cref`，正文结论 ↔ 推导附录双向可追溯；**禁止硬编码章号/式号**。
4. **难度分层**：章节标 `基础 / 进阶 / 选读`，可导航、可跳读。
5. **术语与符号**：前置「记号与约定」+「术语对照（中英）」+ 符号表（`glossaries`），全书一致。

## 三、合并三书的规则

| 角色 | 著作 | 取用 |
|---|---|---|
| **理论骨架**（严谨主线、推导最全） | Barfoot《状态估计》 | 概率/线性高斯/MAP/矩阵李群/连续时间 |
| **直觉 + 代码落地** | 视觉SLAM十四讲 | 几何直觉、可运行要点 |
| **广度 + 现代视角 + 前沿** | SLAM Handbook | 因子图、各模块 SOTA、前沿连接 |

- 同一知识点多视角 → **并列保留**，标注「谁更严谨 / 谁更直觉」。
- **约定冲突 → 统一为全书约定**（见 §五），引用异制处**显式转换**。
- 真差异（不同方法/严谨度）保留；换皮重复取最优表述，余者 `\cite` 带过。
- 图 → **tikz 重画**（版权合规 + 风格统一），不搬原图。
- **配图体系**：工具选型 / API / 案例 / 可复现管线见 **`docs/绘图手册.md`**；脚手架 `figures/`（`figs.sty` 内联库 + `src/` 源 + `pdf/` 产物 + `build_figures.sh`）。示意图→TikZ(+tikz-3dplot/bayesnet/pgfgantt/timing)、真三维→Asymptote、数据图→matplotlib，统一导 **PDF** 经 `\includegraphics` 入书（`\includegraphics` 不吃 SVG；只读编译前先跑 `build_figures.sh`）。

## 四、每章统一模板

1. **引入 / 动机**：要解决什么问题、为何需要。
2. **直觉**：几何 / 物理图像（tikz 图）。
3. **形式化**：定义 → 定理 / 命题（带证明或证明引用）。
4. **推导**：核心步内联正文；次要/冗长逐步推导入**「§ 章末 推导附录」**。
5. **三书对照**：不同视角 / 约定 / 严谨度并列说明。
6. **算法 / 代码**：`codebox`，可运行要点。
7. **小结 + 练习 + 参考指引**。

## 五、记号与约定基线（全书强制，详见《记号与约定》章）

- 旋转矩阵记 $\mathbf{R}\in\mathrm{SO}(3)$（Barfoot 用 $\mathbf{C}$）；位姿 $\mathbf{T}\in\mathrm{SE}(3)$。
- 四元数：**Hamilton**（与 Eigen / 十四讲一致），引用 JPL 处显式转换。
- **扰动：全书右扰动为主**（$\mathbf{R}\,\mathrm{Exp}(\delta\boldsymbol\phi)$，局部坐标），右雅可比 $\mathbf{J}_r$ 为主；引用 Barfoot 左扰动推导处并列转换。
- 坐标系：统一下标语义（如 $\mathbf{T}_{wc}$ = 相机到世界），主动/被动声明。
- $\mathfrak{se}(3)$：排序 $\boldsymbol\xi=[\boldsymbol\rho;\boldsymbol\phi]$（平移分量在前），且 $\boldsymbol\rho\neq\mathbf{t}$（相差左雅可比 $\mathbf{J}$）。

## 七、论文集成（后续大量加入）

**三种模式（按优先）**：
1. **正文成节（主）**：论文核心内容**直接写成正文小节/章节**，按**主题**组织（定义/定理/推导的学术化形式），`\cite` 标源。一篇重要论文可独占一节甚至一章——读者看到的是成体系的知识，而非"论文罗列"。**组织主线是知识体系（topic-first），论文是素材来源。**
2. **抽取片段（inline）**：论文的某个结果嵌入已有小节，作为一种方法/视角，`\cite` 标注。
3. **精读盒（按需）**：少数需评述/对比/批判时才用 `paper` 盒，固定结构 **问题 → 核心思想 → 关键结果 → 与本书关系 → 局限**。

**放置**：成节/片段 → 就近按主题融入；精读盒 → 章末「代表性工作」节；前沿专题可设独立部。

**文献管理**：
- 统一 `refs.bib`（biblatex + biber）；量大后拆 `refs_books.bib` / `refs_papers.bib`。
- 键名规范 `作者年份关键词`（如 `qin2018vins`）。
- 经 **Zotero（MCP）/ `/academic` 技能** 导出 bibtex、做精读，再抽取入书 —— 形成 论文 → Zotero → 精读 → 集成 流水线。

**纪律**：每条引入结果给出处 `\cite`；重建/改写公式打 `\rebuilt`/`\pz`；跨章去重，同一成果只深讲一次，他处 `\cref` 指向。
