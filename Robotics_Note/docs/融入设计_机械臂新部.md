# 融入设计 · 新「机械臂规划与控制」部

> **本文是设计稿，不含章节正文。** 目标：在《机器人学笔记》P3 规划 与 P4 控制之间，**新建一个串联机械臂专部**，把目前散落在 P3（协作操作）、P4（`面向机器人的控制` 一节）、P8（RL 操作）里的机械臂内容**收口成体系**，并为 P8 的 RL 机械臂章提供经典理论支撑。
>
> 设计纪律：**有机结合、承上启下、复用不造轮子**——李群/刚体取 P0、非线性优化取 P1、轨迹优化思想取 P3、控制工具取 P4，Robot `arm_control` 工程当**实战案例**而非堆砌。可扩展（用户后续补机械臂教材源）。
>
> 调研依据：本设计建立在对接线点文件（`rigid_body_motion.tex`/`lie_theory.tex`/`control_intro.tex`/`prac_manipulation.tex`）、P3 part、`refs.bib`、`Robot吸收_arm机械臂运控.md`、`Handbook_新增文献_待并.bib`、`编写规范.md` 的实读之上。
