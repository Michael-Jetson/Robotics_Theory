## 总体结论

**完备性：A−（研究生级状态估计课程的主干"优化—滤波"双线几乎齐全，且循序渐进、互证严密；缺口集中在非高斯/直方图滤波、信息滤波与平方根滤波这两支"经典但当代 SLAM 弱化"的分支，以及粒子滤波的深度）。**

P1 三章实际覆盖了一门研究生状态估计课的绝大部分主干，且组织得相当成熟：

- **概率基元与贝叶斯递推**：`slam_state_estimation.tex` §2（概率/Bayes/边缘化/期望/独立↔不相关）、`kalman_eskf.tex` §`sec:bayes-filter`（贝叶斯滤波预测–校正、马尔可夫假设、"精确但不可实现"）齐备；母方程在 `intro_slam.tex`（Durrant-Whyte 预测/校正式 `eq:intro-dw-predict/update`）已先行铺垫。
- **高斯滤波族**：KF（MAP 配方 + 高斯条件化两路推导）、EKF（三步构造 + 广义高斯滤波 `prop:ggf` 作"总接口"）、IEKF（→众数）、UKF/SPKF/ISPKF（→均值）、ESKF（三态分解 + 18 维误差态 + 注入/复位 + 右/左扰动对照表）全部到位，且给出**滤波器谱系表** `tab:filter-taxo`（按"过非线性方法 × 是否迭代 × 逼近后验哪个方面"分类）——这正是对标书要求的"谱系对比"。
- **批量 vs 递归 / MAP / 最小二乘 / KF≡MAP / 平滑**：`slam_state_estimation.tex` 把"MAP⇒加权最小二乘""正规方程 = 信息矩阵 = 逆协方差 = Hessian = CRLB""线性高斯下 MAP = 贝叶斯均值（三条等价路径）""KF = 批量 MAP 前向递归""块三对角稀疏来自马尔可夫性""RTS = 前向+后向"逐一证明，缝合极好；这是本部分最强的部分。
- **因子图统一视角**：`sec:est-fg`（因子图定义、MAP=NLS、SLAM 各类因子表、变量消元=稀疏分解、Bayes 树/iSAM2 去向）+ `nonlinear_optimization.tex` 的 BA 箭头/Schur，把"滤波—优化—因子图"三者统一，达到 Handbook/√SAM 的现代水准。
- **可观性、一致性、鲁棒**：可观性矩阵 + gauge/零空间、NEES/NIS（含运动 NIS）、偏置使 KF 过自信定理 + 状态增广可观性三例、M-估计/IRLS/RANSAC、"鲁棒核 = 逆 Wishart 先验 MAP"完整证明、GNC/Barron 自适应核——这部分甚至超出多数课程的标准深度。

**教学质量：A。** 严格遵循 v5.0 范式（前置自测→目标→导航图→前置桥接→"如果跳过会怎样"→动机→反面→历史→理论→陷阱→练习→常见误解→小结速查→故障排查）。三章首尾呼应、伏笔与兑现明确（KF≡MAP、EKF≈优化一次迭代、IEKF=单时刻 GN、Schur=变量消元、白化=采样逆运算），过渡承接是范本级的。每章都有阅读方式分级表、符号表、定理速查表、知识点总表、故障排查表。

## 一、缺失主题

### A. confirmed_missing（全书 grep 确认未覆盖，且属课程标准范围）

1. **直方图滤波 / 网格滤波（grid / histogram filter）与离散贝叶斯滤波**
   - 为何重要：这是《Probabilistic Robotics》第 4 章贝叶斯滤波的**第一个可实现实例**，也是理解"贝叶斯滤波 = 任意置信度表示 + 预测/更新"的最朴素入口（先离散直方图，再连续高斯，再粒子）。它直接服务于栅格定位（Markov localization）、占据栅格建图（与 `intro_slam` 已讲的 occupancy grid 地图天然衔接），并给"非参数滤波"一个最简范例。
   - 现状：全书 `grep "直方图滤波|histogram|网格滤波|grid.based filter"` 零命中。`kalman_eskf.tex` 的 pitfall 把"近似表示"二分为"高斯（KF/EKF/UKF）"与"有限样本（粒子）"，**跳过了离散直方图这一支**。
   - 应落何章节：`kalman_eskf.tex` §`sec:bayes-filter`（贝叶斯滤波"精确但不可实现"pitfall 之后）加一小段，或 §`sec:nonlin-pdf` 谱系表前；至少在 `tab:filter-taxo` 增一行"直方图滤波 / 离散贝叶斯 — 离散化状态空间 — 任意（非参数）"。点到为止即可，与 Markov localization 一并提一句去向。

2. **互补滤波 / 姿态航姿（Mahony、Madgwick）**
   - 为何重要：在惯性姿态估计的**工程实践**里，互补滤波（频域：陀螺高通 + 加速度计/磁力计低通）与 Mahony/Madgwick 是 ESKF 之外最常见的轻量替代；研究生课程常以它作"为什么需要概率滤波"的反面对照（确定性、无协方差、调参靠经验）。ESKF 一节既已面向 VIO/惯导融合，理应至少点名互补滤波作为"低算力姿态估计"的对照点。
   - 现状：`grep "互补滤波|complementary|Mahony|Madgwick"` 全书零命中。
   - 应落何章节：`kalman_eskf.tex` §`sec:filter-vs-opt` 的"何时仍选滤波"insight，或 ESKF 节末"ESKF vs 直接 EKF vs UKF"对照处，补一句"更轻量的姿态估计可用互补滤波（确定性、无不确定度传播），适合超低算力 IMU-only 场景"，给一行去向即可。属 partial_thin 边缘，但因姿态融合是本部分重头，列为 confirmed。

### B. partial_thin（已触及但单薄/缺关键机制，建议补全）

3. **粒子滤波的核心机制：重要性采样、权退化、有效样本数 N_eff、何时重采样**
   - 为何重要：粒子滤波是非高斯/非线性后端的代表方法，对标书（Thrun 第 4 章、Särkkä 第 7 章）均以"重要性采样 → 权更新 → 退化（degeneracy）→ 用 N_eff 触发重采样"为核心；FastSLAM/RBPF（Rao-Blackwellized PF）更是 SLAM 古典时代两大主线之一。
   - 现状：`kalman_eskf.tex` §`sec:nonlin-pdf` 仅给 SIR/bootstrap 三步（采样→似然加权→Madow 重采样）与"高维样本贫化、不优于 CRLB"一句。**缺**：(i) 一般重要性采样与建议分布（proposal）的概念；(ii) 权退化现象与 **N_eff = 1/Σwₖ²** 判据（"每步都重采样"反而加剧贫化）；(iii) Rao-Blackwell 化 / FastSLAM——`grep` 显示 FastSLAM 只在 `intro_slam.tex` 出现过名字，P1 滤波章**未接续**，与 intro 的承诺（"`eq:...` 是 `ch:eskf` 滤波章母方程"）存在轻微脱节。
   - 应落何章节：`kalman_eskf.tex` §`sec:nonlin-pdf` 粒子滤波段扩 2–3 句（N_eff 判据 + proposal 一句 + FastSLAM/RBPF 一行去向）；谱系表脚注或正文呼应 `intro_slam` 的 FastSLAM 提及，闭合伏笔。

4. **信息滤波器（Information Filter, IF）作为一个独立成型的递归算法 / 稀疏扩展信息滤波（SEIF）**
   - 为何重要：IF 是 KF 的信息形式对偶，对标书（Thrun 第 3 章专设 Information Filter，及 SEIF 一章）把它作为"多传感器融合廉价、可表达全局无知 Λ→0"的标准滤波器；它也是理解"为什么 SLAM 后端偏信息形式"的桥。
   - 现状：本部分把信息形式讲得很透（`tab:cov-info`、`prop:info-add`、KF 信息形式更新 `eq:kf-info-P/x`、`insight` 里点了"信息滤波 IF 适合多观测少预测"），但**没有把 IF 作为一组完整的预测–校正递归方程单独列出/命名为算法**（如 Thrun 那样给出信息形式的两步）。读者能从碎片拼出，但缺一个集中落点。
   - 应落何章节：`kalman_eskf.tex` §`sec:kf` 在 `insight`（协方差/信息对偶取舍）后，补一个小框 `\begin{proposition}[信息滤波递归]`，把信息形式预测（需两次逆）与校正（信息相加）并列写齐，并点名"它在 SEIF/稀疏化里被用于保持稀疏"。属轻补。

5. **平方根 / 数值稳健滤波（SRIF、UD 分解、Bierman/Potter、Joseph 形式之外）**
   - 为何重要：Maybeck（本审核对标书之一）以**整章**论平方根滤波与"数值条件数 = κ(A)²"问题，并有名言"平方根滤波应总是优先于标准 KF 递推"。研究生估计课通常要求知道：协方差更新的数值病态、平方根（Cholesky/UD）如何把条件数开方、Joseph 形式保对称正定。
   - 现状：**优化侧**讲透了（`nonlinear_optimization.tex` §`sec:nlopt-solve`：白化、Cholesky vs QR、κ(A) vs κ(A)²、√SAM、SRIF 历史 + Maybeck 名言）；**滤波侧**只有 `kalman_eskf.tex` 的 Joseph 形式 pitfall（`(I−KC)P̌` 失对称）。**缺**：滤波递归本身的平方根实现（SRIF/UD/Bierman），以及"协方差滤波数值上为何不如平方根"在滤波语境的一句话。考虑到 √SAM/SRIF 已在优化侧充分覆盖，这里只需在 ESKF/KF 数值陷阱处补一句"序贯滤波的数值稳健实现见平方根信息滤波（与 §`sec:nlopt-solve` 的 √SAM 同源），UD 分解避免显式开方"，并交叉引用。属 partial_thin 偏轻。

6. **不变卡尔曼滤波（InEKF）/ 群仿射系统的滤波一致性**
   - 为何重要：这是过去十年滤波侧最重要的进展之一（对接 Barfoot 第 2 版新增材料、当代 VIO 一致性研究），用李群对称性从根上解决 EKF 的可观性/一致性退化（FEJ 之外的更优方案）。ESKF 一节已是流形滤波，自然该点名 InEKF 作为"右扰动 + 群结构 ⇒ 一致性"的现代去向。
   - 现状：`grep "InEKF|invariant|不变卡尔曼|群仿射"` 全书仅 `vio.tex` 一处提到"（右）不变卡尔曼滤波"作为去向；P1 滤波章**未提**，与本书"右扰动为主"的主线本可天然衔接却未点破。
   - 应落何章节：`kalman_eskf.tex` ESKF 节末（右/左扰动对照后）或 §`sec:consistency` 末，补一句去向 + 一行引用（与 `vio.tex` 呼应），说明"用群结构（InEKF）可使线性化误差与状态无关，改善一致性"。属前沿点到为止。

## 二、脉络 / 教学法问题

总体脉络极佳，下列均为**可打磨项**，非结构缺陷：

1. **粒子滤波伏笔未闭合（脉络）。** `intro_slam.tex` 明说"非高斯样本 ⇒ Rao-Blackwell 化粒子滤波（FastSLAM），是 `ch:eskf` 滤波章母方程"，但 `kalman_eskf.tex` 的粒子滤波段未提 FastSLAM/RBPF，也未回指 intro。建议在粒子滤波段加一句"这正是 `intro_slam` 提到的 FastSLAM 的核（RBPF：路标用解析高斯、机器人轨迹用粒子）"，闭合跨章伏笔。

2. **贝叶斯滤波的非参数分支被结构性略过（脉络 + 完备性）。** `sec:bayes-filter` 的 pitfall 把实现路径二分为"高斯 / 粒子"，使读者形成"贝叶斯滤波只有这两条出路"的错觉，恰好漏掉直方图滤波（见缺失主题 1）。补上直方图一支会让"从精确贝叶斯到各实用滤波"的下降链更完整（离散→高斯→样本），也更贴合 Thrun 的讲法。

3. **信息滤波"有名无方程"（教学法）。** 信息形式被反复使用却未作为独立算法集中呈现（见缺失主题 4）。对初学者，"IF 是什么"散落在 KF 推导、对偶表、insight 三处；一个集中的命题框会显著降低拼图成本。

4. **ESKF 节与谱系/一致性节的衔接可再点一句（脉络）。** ESKF（§`sec:eskf`）紧跟"高斯过非线性三办法 + 谱系"（§`sec:nonlin-pdf`）与"一致性/偏置"（§`sec:consistency`），但 ESKF 自身**未回收**前面的一致性语言（如"误差态恒在原点 ⇒ 线性化有效 ⇒ 一致性优于直接 EKF"已在 insight 提了一半，可与 NEES、InEKF 串一句）。当前 `tab:eskf-pert`、注入/复位讲得很细，唯独"ESKF 对一致性的贡献"与 §`sec:consistency` 的 NEES 框架未显式挂钩。

5. **`nonlinear_optimization.tex` 的 RANSAC 与迭代次数公式在两章重复（教学法/冗余）。** RANSAC 五步 + 迭代次数公式 `k=ln(1−p)/ln(1−wⁿ)` 在 `kalman_eskf.tex` §`sec:filter-vs-opt`（`eq:eskf-ransac`）与 `nonlinear_optimization.tex` §`sec:nlopt-robust`（`eq:nlopt-ransac`）**各写了一遍**，连习题数值（w=0.5,n=2,p=0.99）都重复。这在"自包含"原则下可接受，但建议其一改为交叉引用 + 只留差异，避免读者疑惑"是不是同一个东西"。鲁棒核 + IRLS + 逆 Wishart 也有类似双写（kalman §filter-vs-opt 的 `thm:robust-iw` vs nlopt §robust 的完整证明）——后者更详，前者可压缩为指针。

6. **难度分层标注个别偏保守（教学法，微）。** 如 §`sec:est-batch-recursive`（批量↔递归等价 + 块三对角）标"次优 ★★★"，但其中"KF=批量 MAP 前向递归"是与下一章缝合的**骨干**结论；导航图已把它当骨干，但阅读方式表把它划入"速读可跳"。两处口径可统一（倾向把核心结论留在骨干）。

7. **`rebuilt` 标注一处（编辑痕迹，需清理）。** `nonlinear_optimization.tex` §`sec:nlopt-cov`（约 1055 行）的 CRLB 与 Laplace 关系处出现一个 `\rebuilt{...}` 命令（"CRLB=费舍尔信息之逆为标准结果…此处按经典结果补全"），像是写作期的内部批注/自定义宏。应确认该宏在 `preamble` 有定义且渲染得体，否则清理为正文脚注或常规叙述。

## 三、独立性问题

整体独立性优秀（自有口吻 + 平衡多书对照 + 显式跨书记号词典）。仅以下为**轻微 ventriloquize / 措辞收口**项，无 external_punt 硬伤：

1. **少量"某书指出/某书路线"式署名（ventriloquize，轻）。** 例：
   - `kalman_eskf.tex`：招牌推导一/二命名为"十四讲路线""Barfoot 路线"；"Barfoot §4.2.6 证明 IEKF 收敛到…""（Solà §7 指出全局角误差某些性质更好）"。
   - `slam_state_estimation.tex`：本节"摘自 Barfoot 第 1 章（其内部式号为 2.x）""Barfoot 把这写成…并给三点评注"。
   - 这些**多数已是"对照印证"而非"代替作者发言"**（本书自己把推导写全了，署名只为溯源），属可接受范围。但"Barfoot §4.2.6 证明…""Solà §7 指出…"这类**把结论的责任外包给原书章节号**的句子，建议改为本书口吻陈述结论 + 末尾 `\cite` 溯源（如"IEKF 迭代收敛到后验众数（即 MAP）\cite{barfoot2024state}"），与本书"自包含、读者无需翻原书"的总纲一致。

2. **Maybeck 名言直引（narration，极轻）。** §`sec:nlopt-solve` 直引 Maybeck"许多实践者有充分理由主张平方根滤波应总是优先于…"。作为历史点缀可保留，但这是本部分唯一一处"借名人之口下论断"；若要更纯本书口吻，可改为转述 + 引用。

3. **跨书记号词典是独立性的范本（正面记录）。** `slam_state_estimation.tex` 与 `kalman_eskf.tex`、`nonlinear_optimization.tex` 的记号区把 Q/R、v/n、ρ≠t、J 转置、Cholesky 上/下三角的三书差异逐条登记并统一到本书符号——这正是"平衡多书对照 + 本书口吻"的正确做法，应作为其他章模板。
