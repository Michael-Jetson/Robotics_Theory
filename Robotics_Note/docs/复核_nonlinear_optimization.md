### 2.1 ⭐ `eq:lm-rho-res`（对抗式头号核对项）—— **正确，复核无误**

当前文本（line 428–431）：
```
ρ = [J(x) − J(x⊞Δx)] / [ −(∇J)^⊤ Δx ]   （分子、分母均为标量目标的下降量；下降步使 (∇J)^⊤Δx<0）
```
- **分子** `J(x)−J(x⊞Δx)`：标量目标的**实际下降量**。✓ 标量。
- **分母** `−(∇J)^⊤Δx`：一阶模型 `L1(Δx)=J(x)+(∇J)^⊤Δx` 的**预测下降** `L1(0)−L1(Δx)`。✓ 标量。`∇J=J^⊤W^{-1}e`（与 `eq:gn-normal` 右端"梯度的相反数"、附录 line 1359 一致）。下降步 `(∇J)^⊤Δx<0` ⇒ 分母 >0，与分子同号，ρ>0。
- **判定**：这是标准的"实际下降/一阶预测下降"标量增益比，**不再是**审计/原始待办记录的"向量/向量"错误形式（旧错：分子 `f(x+Δx)−f(x)`、分母 `J^⊤Δx` 皆向量）。**之前发现的瑕疵已被正确修复。**
- 与十四讲原式 (6.34) 的关系（值得肯定的处理）：原书 (6.34) 是 `[f(x+Δx)−f(x)]/[J(x)^⊤Δx]`（分子分母同为负、比值正，且 `f`/`J` 指标量目标 `F` 与其梯度，OCR/原书记号易致歧义）。本书**没有照搬**原式，而是改写为更干净、无歧义的"+下降/+下降"标量形式并 `\cite` 原书——数学等价、表述更优。**正确处理。**
- WebSearch 交叉验证（Madsen–Nielsen / NEOS）：ρ = 实际下降 / 模型预测下降，二者皆标量；标准式（`eq:lm-rho`）用二次模型 `L(0)−L(Δx)` 作分母，本式用一阶预测作分母——均为合法变体。**一致。**
